# MERN Stack Task 4 - JWT Authentication System

A simple Login and Signup Authentication System built using the MERN Stack and JWT Authentication.

## Features

* User Signup
* User Login
* Password Hashing using bcrypt
* JWT Token Authentication
* Protected Login State
* Logout Functionality
* Attractive Animated UI

## Tech Stack

* React.js
* Node.js
* Express.js
* MongoDB
* JWT
* bcryptjs

## Run Backend

```bash id="r4m8qx"
cd backend
npm install
node server.js
```

## Run Frontend

```bash id="p7v2kt"
cd frontend
npm install
npm start
```
